MQ135 Pollution Detection Sensor

A compact air-quality monitor built with MQ135, Arduino Uno and a 16x2 LCD — learn how it works and how to build it.

See Wiring View Code

What is this project?

This project measures air quality (volatile compounds and gas concentration) using the MQ135 sensor. The sensor's analog output is read by an Arduino Uno and the measured value is shown on a 16x2 LCD. This demo site documents the components, wiring, code and assembly.

Components

  • MQ135 gas sensor module
  • Arduino Uno
  • 16x2 LCD (Hitachi HD44780 compatible)
  • Breadboard and jumper wires
  • USB cable and 5V power

What you will learn

  • How to wire MQ135 to Arduino and LCD
  • How to read analog values and display them
  • Basic calibration notes and interpretation

Wiring & Assembly

Connect the MQ135 analog output to an analog pin (A0) on the Arduino. If you use an MQ135 module it will have VCC (5V), GND and AO pins. Connect the 16x2 LCD using the standard 4-bit wiring with a potentiometer to adjust contrast.

MQ135 → Arduino

  • MQ135 VCC → 5V
  • MQ135 GND → GND
  • MQ135 AO → A0

16x2 LCD → Arduino (4-bit)

  • LCD RS → D12
  • LCD E → D11
  • LCD D4 → D5
  • LCD D5 → D4
  • LCD D6 → D3
  • LCD D7 → D2
  • LCD VSS → GND, VCC → 5V, V0 (contrast) → pot wiper

Arduino Sketch

Below is the sketch used with the MQ135 sensor and 16x2 LCD. The code is included inline on the page so you can copy it directly.

#include <Wire.h>
#include <LiquidCrystal_I2C.h>

#define MQ135_PIN A0   // MQ135 sensor connected to analog pin A0
#define BUZZER_PIN 8   // Buzzer connected to digital pin 8

int airQuality = 0;     // Variable to store sensor reading
int threshold = 25;    // Pollution threshold value (adjust based on testing)

// Create LCD object (try 0x27 first; if nothing shows, use 0x3F)
LiquidCrystal_I2C lcd(0x27, 16, 2);

void setup() {
  Serial.begin(9600);
  pinMode(BUZZER_PIN, OUTPUT);
  digitalWrite(BUZZER_PIN, LOW);
  
  lcd.init();           // Initialize the LCD
  lcd.backlight();      // Turn on LCD backlight
  lcd.setCursor(0, 0);
  lcd.print("Air Quality Mon");
  lcd.setCursor(0, 1);
  lcd.print("Initializing...");
  delay(2000);
  lcd.clear();
}

void loop() {
  airQuality = analogRead(MQ135_PIN);  // Read analog value from MQ135
  Serial.print("Air Quality Value: ");
  Serial.println(airQuality);

  lcd.setCursor(0, 0);
  lcd.print("AQ Value: ");
  lcd.print(airQuality);
  lcd.print("   "); // Clear trailing digits

  // Check if pollution exceeds threshold
  if (airQuality > threshold) {
    lcd.setCursor(0, 1);
    lcd.print("Status: POLLUTED ");
    digitalWrite(BUZZER_PIN, HIGH);
  } else {
    lcd.setCursor(0, 1);
    lcd.print("Status: CLEAN    ");
    digitalWrite(BUZZER_PIN, LOW);
  }

  delay(1000); // 1-second update rate
}
